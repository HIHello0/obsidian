 # Obsidian Short cut   
 --- 
 ##    
 [[obsidian]]

- ctrl + o -> search bar   
- ctrl + p -> command [search]   
- ctrl + right click → video note side by side   
- [[...#]] → link specific block in page   
- ctrl + G → graph view   
- ctrl + shift + n  → New note in plan mode   
- ctrl + e → preview mode (switch mode)   
- ctrl + right click → to preview link
